# Markdown 的基础语法超详细（2）

part6. 列表

Ⅰ 无序列表
所需格式：<输入’8 上面的星星号’加空格> 或 < 输入 + 加空格 > 或 < 输入 - 加空格 > 写入内容后回车
* 示例: *
①

  • 我想是一
  • 我想是二
  • 我想是三（星星开头）
  • 重复一次
  • 重复两次
  • 重复三次（加号开头）
  • 再来一遍
  • 再来两遍
  • 再来三遍

tips : 没有空格，效果不会出发哦！(⊙﹏⊙)

Ⅱ 有序列表
所需格式 : <输入 1.> 写入内容后回车
* 示例: *
①
1. 是一
2. 是二
3. 是三
②

  1. 是一
  2. 是二
  3. 是三
    ⑤
  4. 是一
  5. 是二
  6. 是三

tips：有序的列表没有空格的需求，但是没有空格是，形式会很奇怪

tips : 后期会有进阶讲列表升级（一级列表、二级列表）

part7. 任务列表
所需格式：<- [] > 与 <- [x] > 注意’]' 后有空格
* 示例: *
①

part8. 表格
所需格式：<每行都是‘| 内容 | 内容 |’（第二行要 |—|---|…）> 内容间用‘|’分隔，没有空格要求，但要在两侧加上‘|’围边
* 示例: *
①

表格 表格
表格 表格
表格 表格

part9. 粗体
所需格式：<’8 上的星星连续 2 个‘内容’8 上的星星连续 2 个‘>，不是很好表达
* 示例: *
①cuti ②粗体

tips：它也是可以嵌套地哦！

part10. 斜体
所需格式：<’8 上的星星 1 个‘内容’8 上的星星 1 个‘>，不是很好表达
示例: *
①xieti* ②斜体
